Multiple Plots in One Figure (Subplots)

  • Subplots allow you to place multiple plots in a single figure.

  • Managed through the subplot function or the subplots function for more flexibility.

Demo data

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 2*np.pi, 100)
y = np.cos(x)
z = np.sin(x)

Creating Subplots Using subplots: code

fig, ax = plt.subplots(1, 2) # 1 row, 2 columns

ax[0].plot(x, y, 'r--') # Plot on the first axis
ax[1].plot(x, z, 'g*-') # Plot on the second axis

plt.show()

Using the the squeeze argument

  • The squeeze argument in plt.subplots() controls whether a single Axes object or an array of Axes objects is returned when creating a single subplot.

  • By default, squeeze=True. This means:

  • If the subplot layout results in a single subplot (e.g., plt.subplots(1, 1)), a single Axes object is returned.

  • For layouts with more than one subplot, an array of Axes objects is returned.

  • Setting squeeze=False forces the function to always return an array of Axes objects, even if there’s only one subplot.

Single Subplot Without Squeeze: code

# Returns a single Axes object by default
fig, ax = plt.subplots(1, 1) 
ax.plot(x, y)
plt.show()

Single Subplot Without Squeeze: plot

Single Subplot With squeeze=False: code

# Returns an array containing a single Axes object
fig, axs = plt.subplots(1, 1, squeeze=False) 
axs[0, 0].plot(x, y)
plt.show()

Customizing Subplots: code

# Share axes with `sharex` and `sharey`
fig, ax = plt.subplots(2, 2, sharex='col', sharey='row')

# Adjust spacing with `plt.subplots_adjust`.
fig.subplots_adjust(hspace=0.5, wspace=0.5) # Adjust spacing

# Add titles and labels to each subplot
ax[0, 0].plot(x, y)
ax[0, 0].set_title('Top Left')

ax[0, 1].plot(x, z)
ax[0, 1].set_title('Top Right')

ax[1, 0].plot(y, z)
ax[1, 0].set_title('Bottom Left')

ax[1, 1].plot(x, x)
ax[1, 1].set_title('Bottom Right')

Adding Legends: code

# use the `label` parameter of `plot`
plt.plot(x, y, '-b', label='Sine')

plt.legend(loc='upper right')  # Specifies the location of the legend
plt.xlabel('X axis')
plt.ylabel('Y axis')
plt.title('Sine and Cosine Functions')
