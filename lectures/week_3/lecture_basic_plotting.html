Creating Simple Line Plots: code

# Import Matplotlib's pyplot
import matplotlib.pyplot as plt
import numpy as np

# Generate some data
x = np.linspace(0, 10, 100)
y = np.sin(x)

# Create a line plot
plt.plot(x, y)
plt.show()

  • Plot the function \(y = \cos(x)\) for \(x\) ranging from 0 to \(2\pi\).

  • Use a blue dashed line for the plot.

  • Mark each data point with a red circle.

  • Add a title to the plot: “Cosine Wave”.

  • Label the x-axis as “X” and the y-axis as “Y”.

  • Set the x-axis limits from 0 to \(2\pi\) and the y-axis limits from -1.5 to 1.5.

  • Add a legend with the label “cos(x)” in the upper right corner

  • Add grid lines

Answer: code

import matplotlib.pyplot as plt
import numpy as np

# Data
x = np.linspace(0, 2*np.pi, 100)
y = np.cos(x)

# Plot
plt.plot(x, y, linestyle='--', color='blue', marker='o',
            markerfacecolor='red', label='cos(x)')
plt.title('Cosine Wave')
plt.xlabel('X')
plt.ylabel('Y')
plt.xlim(0, 2*np.pi)
plt.ylim(-1.5, 1.5)
plt.legend(loc='upper right')
plt.grid(True)
plt.show()
